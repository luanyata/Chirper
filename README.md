# Twiter Clone 

Projeto simula algumas funcionalidades da rede social [Twitter](twitter.com).
O projeto faz parte do curso Desenvolvedor React da Udacity.

## Objetivo
Por em pratica o conhecimento sobre gerenciamento de estado previsivel com React e Redux.

## Funcionalidades

* Timeline
* Criar Tweet
* Curtir Tweet
* Responder Tweet


## Dependencias de Desenvolvimento

* node
* yarn


## Dependencias do Projeto
   
* react
* react-dom
* react-icons
* react-router-dom
* react-scripts
* redux
* redux-thunk
* react-redux
* react-redux-loading


## Instalando Dependencias do Projeto
``` 
$ yarn install
``` 

## Rodando o Projeto
``` 
$ yarn start
```
